# React Starter

Starter kit for React applications.

## Features

- [x] ES6 support
- [x] [React](http://facebook.github.io/react/)
- [x] [Webpack v3](https://webpack.github.io)

## Setup

```bash
$ npm install
```

## Usage

Start webpack dev server:

```bash
$ npm start
```
